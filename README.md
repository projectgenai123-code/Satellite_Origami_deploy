# 🛰️ Satellite Origami Designer

AI-powered satellite origami crease pattern generator.

## Features
- CVAE generative model trained on ETH Zurich 8M origami dataset
- BM25 RAG chatbot for satellite Q&A
- 3D interactive viewer (Three.js)
- Custom satellite configuration (solar panels, antenna, reflector, truss)

## Tech Stack
- Backend: Flask + PyTorch
- Frontend: HTML/CSS/JS + Three.js
- AI: CVAE + BM25 RAG
